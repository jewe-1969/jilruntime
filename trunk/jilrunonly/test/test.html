The purpose of this example HTML file is to demonstrate the "urlfilter.jc" script file. Just pass the name of this HTML file to the urlfilter.jc script on the command-line, like this:

      jilrunonly urlfilter.jc test.html

Similar projects

There are a number of similar virtual machine / script engine projects in development and available under various licenses. Most of them are much longer in development than JewelScript, and thus more advanced.

ProjectLicenseDescription
AngelScriptopen source, zlib/libpngAn extremely flexible cross-platform scripting library designed to allow applications to extend their functionality through external scripts.
EELopen source, GNU LGPLA scripting and programming language, designed specifically for hard real time applications.
GameMonkey Scriptopen source, MIT licenseAn embedded scripting language that is intended for use in game and tool applications.
Lua 5.0open source, MIT licenseA powerful light-weight programming language designed for extending applications.
Pythonopen source, GPL-compatibleAn interpreted, interactive, object-oriented programming language. It is often compared to Tcl, Perl, Scheme or Java.
Squirrelopen source, zlib/libpngA high level imperative/OO programming language, designed to be a powerful scripting tool that fits in the size, memory bandwidth, and real-time requirements of applications like games.
